## Introduction

ReadMEasy is a README generator made with Vue.js, Bootsrap, SASS and Firebase, for data base, hosting and authentication.

You can find it at https://readmeasy.web.app and it is also installable as PWA with the menu button or manually.

## Installation

1\. Clone Repository:

```bash
git clone https://github.com/pablovacasgarcia/ReadMEasy
```

2\. Navigate to the project directory:

```bash
cd ReadMEasy
```

3\. Install the dependencies:

```bash
npm install
```

4\. Add firebase confg:

Add your Firebase configuration called 'firebase' to the main root

## Usage

To run the project locally, use the following command:

```bash
npm run serve
```

Open your browser and navigate to http://localhost:8080 to see the application in action.

## Features

*   ReadME Workbench: For creating and editing ReadMEs
*   Use templates: Use template to create ReadMEs fast and easily
*   Explore: Explore other users profile and ReadMEs
*   Follow and Like: Follow users and Like ReadMEs, to reach them easily

## Example

This README is actually done with ReadMEasy and here is another example of a README made with ReadMEasy:

```python
# How to create a README

Creating a README could be really easy or diffcult. Which way will you choose?

## Difficult Way ❌

1.  Learn MarkDown Language 📖
2.  Think about what to write 💭
3.  Waste your time testing ⌚
4.  Give up and leave it halfway 😿
5.  Back to step 1 ⤴︎

## ReadMEasy Way ✔️

1.  Open ReadMEasy 🌐
2.  Choose a tempate 👍
3.  Edit it to your liking ✎
4.  Download it and upload it to your repository 📂
5.  Enjoy 👇🏻
```
